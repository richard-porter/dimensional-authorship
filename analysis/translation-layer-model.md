# Translation Layer Model

In human–AI collaboration, two distinct cognitive layers must be separated:

1. Constraint Architecture (AI-facing)
- Binary gates
- Enforcement logic
- Failure detection
1. Pattern Preservation (Human-facing)
- Meaning continuity
- Voice integrity
- Boundary articulation

High-quality collaboration requires a translation loop between these layers.

Governance failures often occur when constraint design replaces pattern preservation, or when pattern preservation ignores constraint mechanics.

This case study documents the need for both.

*In the Richard Porter AI Safety Ecosystem, the three-part implementation of this translation loop is: the [Frozen Kernel](https://github.com/richard-porter/frozen-kernel) (constraint architecture layer), [ROSETTA](../experiments/dimensional-voice-stamp-v01.md) (pattern preservation layer), and the [Human Sovereign Collaboration Framework](../experiments/human-sovereign-collaboration-framework.md) (the translation loop between them).*
